# TencentIM.Native 使用指南

## 1. 基础使用

### 初始化和登录

```csharp
using TencentIM.Native;
using TencentIM.Native.Utils;

// 创建客户端
var client = new TencentIMClient(SDK_APP_ID);

// 初始化SDK
if (!client.TIMInit())
{
    Console.WriteLine("初始化失败");
    return;
}

// 生成UserSig
var sigGenerator = new UserSigGenerator(SDK_APP_ID, SECRET_KEY);
var userSig = sigGenerator.Generate(USER_ID);

// 登录
if (await client.TIMLogin(USER_ID, userSig))
{
    Console.WriteLine("登录成功");
}
```

### 接收消息

```csharp
// 设置消息接收事件（在TIMInit后自动注册）
client.OnMessageReceived += (sender, e) =>
{
    Console.WriteLine($"收到消息: {e.SenderId} - {e.Content}");
};

client.OnConnected += (s, e) => Console.WriteLine("已连接");
client.OnDisconnected += (s, e) => Console.WriteLine("已断开");
```

## 2. 发送消息

发送消息需要构造JSON格式的消息参数：

```csharp
using System.Text.Json;
using TencentIM.Native.Models;

// 发送文本消息
var textMsg = JsonSerializer.Serialize(new
{
    message_elem_array = new[]
    {
        new
        {
            elem_type = (int)TIMElemType.Text,
            text_elem_content = "Hello World"
        }
    }
});

await client.TIMMsgSendMessage("user_id", TIMConvType.C2C, textMsg);
```

## 3. 群组管理

```csharp
// 创建群组
await client.TIMGroupCreate("我的群组", "Public");

// 加入群组
await client.TIMGroupJoin("group_id", "你好");

// 邀请成员
await client.TIMGroupInviteMember("group_id", new[] { "user1", "user2" });

// 删除成员
await client.TIMGroupDeleteMember("group_id", new[] { "user1" });

// 获取已加入群组列表
await client.TIMGroupGetJoinedGroupList();

// 退出群组
await client.TIMGroupQuit("group_id");

// 解散群组
await client.TIMGroupDelete("group_id");
```

## 4. 好友管理

```csharp
// 添加好友
await client.TIMFriendshipAddFriend("user_id", "备注名");

// 删除好友
await client.TIMFriendshipDeleteFriend("user_id");

// 获取好友列表
await client.TIMFriendshipGetFriendProfileList();
```

## 5. 会话管理

```csharp
// 获取会话列表
await client.TIMConvGetConvList();

// 删除会话
await client.TIMConvDelete("conv_id", TIMConvType.C2C);
```

## 6. 登出和清理

```csharp
// 登出
await client.TIMLogout();

// 释放资源
client.Dispose();
```
