# TencentIM.Native - 使用说明

## 项目结构

```
05-程序模块/
├── TencentIM.Native/              # C SDK封装库
│   ├── Interop/                   # P/Invoke声明
│   │   ├── TIMNative.cs          # 基础API
│   │   ├── TIMNative.Message.cs  # 消息API
│   │   └── TIMNative.Group.cs    # 群组API
│   ├── Models/
│   │   └── TIMEnums.cs           # 枚举定义
│   ├── TencentIMClient.cs        # 主客户端
│   ├── TencentIMClient.Message.cs # 消息功能
│   ├── TencentIMClient.Group.cs   # 群组功能
│   └── native/                    # DLL存放位置
│       └── x64/
│           └── ImSDK.dll          # 需要手动放置
└── TencentIM.Native.Test/         # 测试项目
    └── Program.cs
```

## 使用步骤

### 第1步：下载C++ SDK

1. 访问：https://github.com/tencentyun/TIMSDK
2. 下载Windows版本
3. 找到 `ImSDK.dll`（64位版本）

### 第2步：放置DLL

将 `ImSDK.dll` 复制到：
```
05-程序模块/TencentIM.Native/native/x64/ImSDK.dll
```

### 第3步：配置测试

编辑 `TencentIM.Native.Test/Program.cs`：
```csharp
const long SDK_APP_ID = 你的应用ID;
const string USER_SIG = "你的UserSig";
```

### 第4步：运行测试

```bash
cd 05-程序模块/TencentIM.Native.Test
dotnet run
```
